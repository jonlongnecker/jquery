Background & Logos

The background colors and logos are set in the base.css file of the parent theme and are triggered by id's on the body. Unless you're setting up a site with a new color scheme and logo you should just need to use one of the following id's.

  • jQuery: No ID needed. (blue)
  • jQuery UI: #body-jquery-ui (orange)
  • jQuery Mobile: #body-jquery-mobile (green)
  • Sizzle Js: #body-sizzlejs (dark orange)
  • Qunit Js: #body-qunitjs (forest green)
  • jQuery Foundation: #body-jquery-foundation (aqua)

If you are adding a site with a new logo/color scheme, please add to base.css Don't forget the retina version of the logo and the media query to toggle it based on pixel density.

Other colors (like headers, buttons & links) and layout specific to the site will be in the style.cssof that site's theme.

Foundation

Please use this basic foundation for all pages.

  • The #container sets the background color and provides padding around the white content area.
  • The #content-wrapper creates the white content area and gives the appropriate padding.
  • The #content should be used for the main content column.
  • The #sidebar should be used for the secondary column or sidebar.
<body>
  <header>
    Header (Global Nav, Logo and Main Nav)
  </header>
  <div id="container">
    <div id="content-wrapper" class="clearfix">
      <div id="content">
        Content goes here
      </div>
      <div id="sidebar">
        Sidebar Here
      </div>
    </div> <!--End Content Wrapper-->
  </div> <!--End Container-->
</body>

Embeds

For an embedded element like a YouTube or Vimeo video, iframe embed, etc.. make sure you wrap in a embed div so that it scales proportionately with media queries.

YouTube Embed

</div>
